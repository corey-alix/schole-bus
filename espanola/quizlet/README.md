Vocabulary tester with speech synthesis, user required to press F1 to active Probably only works in chrome

Game Concept:

-   Spanish 101 (or earlier) for English speakers
-   input: keyboard and/or voice
-   output: screen and/or audio
-   Progression bases on word count
-   Reach minimal threshold to unlock additional nouns and verbs
-   Reach minimal threshold to unlock longer phrases
-   Proficency in one language pattern unlock new patterns (new tenses, pronouns)
-   Language patterns include tenses, additional pronouns (not sure what they all are)
